使用border-radius属性制作神奇背景效果

发表于 2017-06-01

马上就要毕业了，投了很多份简历都石沉大海了，这两天杭州的天气闷闷的，静坐在阳台上都出了一身汗，最近一直在忙着工作的事情，烦了，写点东西玩玩，缓解缓解郁闷的心情。

之前就见过这种效果的背景，只是觉得很好玩，并没有去研究实现思路，今天下午坐在电脑前，想了想，可以使用border-radius和伪类before after来实现。

大致思路

利用伪元素after,before构成层级，z-index控制位置以及border-radius和过渡来实现

html结构

<body>
    <ul class="mainul">
        <li class="son">JoseLee</li>
        <li class="son">Woffee</li>
        <li class="son">EchoChen</li>
        <li class="son">AnitaMui</li>
        <li class="son">DulyWang</li>
    </ul>
</body>

css代码

基础

.mainul > .son {
        position: relative;
        overflow: hidden;
        z-index: 1;
        float: left;
        margin-left: 30px;
        padding: 0 10px;
        height: 50px;
        color: #ffffff;
        font-size: 600;
        line-height: 50px;
        list-style: none;
    }
after
.mainul > .son:after {
        background-color: #199475;
        border-radius: 200px;
        bottom: 0;
        content: " ";
        height: 0;
        left: 0;
        margin: auto;
        position: absolute;
        right: 0;
        top: 0;
        transition: all 0.3s ease 0s;
        width: 0;
        z-index: -2;//设置为最底层
    }

:hover:after

.mainul > .son:hover:after {
        width: 100%;
        height: 100%;
        border-radius: 5px;
        left: 0;
    }

before

.mainul > .son:before {
        background-color: #0b6e48;
        border-radius: 200px;
        bottom: 0;
        content: " ";
        height: 0;
        left: 0;
        margin: auto;
        position: absolute;
        right: 0;
        top: 0;
        transition: all 0.3s ease 0.1s; //延迟0.1S执行，显示出层次感
        width: 0;
        z-index: -1; // 设置为倒数第二层
    }
